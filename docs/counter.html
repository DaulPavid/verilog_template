# Counter Example Specification ## Overview This is an example counter which serves as a filler module for the template. ## IO Ports There are only a few ports on this module because it's just a simple counter. Port | Direction | Bus Width | Description ---------|:----------|:----------|:------------- i_clk | Input | 1 | Counter clock i_rst | Input | 1 | Counter reset o_count | Output | 16 (VAR) | Counter output register ## Registers The counter has no runtime configuration registers.
